RestController API Documentation

Integrantes: Javier Garcia, Armando Montiel, Leandro Rúa

/api (GET)

Description: Returns a simple greeting "Hello, world!".

/api/allUsers (GET)

Description: Returns a list of all users stored in the database.

/api/allNotes (GET)

Description: Returns a list of all notes stored in the database.

/api/Ubyid/{userId} (GET)

Description: Returns a specific user based on the provided ID.

/api/Nbyid/{noteId} (GET)

Description: Returns a specific note based on the provided ID.

/api/UpdateUbyid/{userId} (PUT)

Description: Updates the information of a specific user based on the provided ID.

/api/UpdateNbyid/{noteId} (PUT)

Description: Updates the information of a specific note based on the provided ID.

/api/newuser (POST)

Description: Inserts a new user into the database.

/api/newnote (POST)

Description: Inserts a new note into the database.

/api/login (POST)

Description: Authenticates a user based on the provided username and password, and generates a JWT token for the authenticated user.

/api/DeleteUbyid/{userId} (DELETE)

Description: Deletes a specific user from the database based on the provided ID.

/api/DeleteNbyid/{noteId} (DELETE)

Description: Deletes a specific note from the database based on the provided ID.
